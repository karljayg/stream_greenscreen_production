How to Use

To use the production tool, follow these steps:

  1. Download the project files from GitHub.
  2. Open the index.html file in your web browser.
  3. Enter the name of the player you want to display in the input box.
  4. Select the media file you want to play from the dropdown list.
  5. Click the Play button to start playing the media file.

Supported Media Files

The production tool supports the following media files:

  • Video files: .mp4
  • Audio files: .mp3
  • Image files: .jpg, .jpeg, .png, .gif
